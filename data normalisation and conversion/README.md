# **Experiment 14A: Data Normalization**

## **Aim**

The aim of this experiment is to understand and implement different data normalization techniques using Python.

---

## **Description**

In this experiment, I created a dataset containing product details such as price, units sold, and discount. I applied various normalization techniques to scale the data into a suitable range for analysis.

---

## **Techniques Used**

### **Min-Max Normalization**

I used Min-Max normalization to scale values between 0 and 1. This method was applied to price, discount, and units sold columns.

### **Z-Score Normalization**

I applied Z-score normalization to the units sold column to standardize the data based on mean and standard deviation.

### **Decimal Scaling**

I used decimal scaling to reduce large values by dividing them with powers of 10. This was applied to price and discount.

### **Normalization of Multiple Columns**

I also normalized multiple columns together using Min-Max normalization to make the dataset consistent.

---

## **Result**

After applying all normalization techniques, the dataset values were scaled properly. This makes the data easier to analyze and suitable for machine learning models.
