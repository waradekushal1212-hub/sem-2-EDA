# Tools for EDA Study of NumPy Library

---

## Overview

Exploratory Data Analysis (EDA) is the process of examining datasets to summarize their main characteristics using statistical techniques and data inspection methods. The NumPy library plays a fundamental role in EDA by providing powerful numerical computation tools and efficient multi-dimensional array handling in Python.

NumPy enables fast mathematical operations, statistical analysis, and structured data manipulation, making it a core component of data science workflows.

---

##  Aim

To study and understand the tools provided by the NumPy library for performing Exploratory Data Analysis (EDA), and to learn how its array structures and mathematical functions support efficient data exploration.

---

## 🛠 Tools Used

1. Python – Programming language used for implementation.
2. NumPy – Core numerical computation library for array operations and statistical analysis.
3. Jupyter Notebook – Interactive environment for writing and executing code.
4. Matplotlib – Optional library for data visualization.
5. Pandas – Support tool for handling structured datasets.

## 📚 Theory and Key Concepts

### 1️⃣ NumPy Arrays (ndarray)

The primary data structure in NumPy is the *ndarray* (N-dimensional array), which supports:

- Multi-dimensional data storage
- Homogeneous data types
- Efficient memory usage
- Fast vectorized operations

---

### 2️⃣ Statistical Functions for EDA

NumPy provides built-in statistical functions that help summarize datasets:

- `mean()` – Calculates average value
- `median()` – Finds middle value
- `std()` – Computes standard deviation
- `var()` – Calculates variance
- `min()` – Identifies minimum value
- `max()` – Identifies maximum value
- `sum()` – Computes total of elements

These functions are essential for understanding data distribution and variation.

---

### 3️⃣ Data Manipulation Concepts

NumPy supports several important EDA operations:

- **Indexing and Slicing** – Accessing specific elements or subsets
- **Reshaping** – Changing array dimensions
- **Sorting** – Arranging elements in order
- **Unique Values** – Identifying distinct elements
- **Boolean Masking** – Filtering data using conditions
- **Random Number Generation** – Creating sample datasets for testing
