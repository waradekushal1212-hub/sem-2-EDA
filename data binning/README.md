# README – Functions Used in Experiment 13

## **Overview**

This document explains the Python and pandas/numpy/matplotlib functions used in the notebook for data analysis and visualization.

---

## **pd.read_csv()**

**Purpose:**
Loads dataset from a CSV file into a DataFrame.

**Usage:**
Used as the first step to import data for analysis.

---

## **pd.DataFrame()**

**Purpose:**
Creates a DataFrame from structured data.

**Usage:**
Used for organizing and manipulating data.

---

## **head()**

**Purpose:**
Displays the first few rows of the dataset.

**Usage:**
Helps in understanding the structure of the data.

---

## **info()**

**Purpose:**
Provides a summary of the dataset.

**Usage:**
Displays column names, data types, and null values.

---

## **describe()**

**Purpose:**
Generates statistical summary.

**Usage:**
Used for mean, median, standard deviation, etc.

---

## **isnull()**

**Purpose:**
Detects missing values.

**Usage:**
Returns boolean values indicating null entries.

---

## **sum()**

**Purpose:**
Calculates total values.

**Usage:**
Used with isnull() to count missing data.

---

## **fillna()**

**Purpose:**
Fills missing values.

**Usage:**
Replaces null values with mean, median, or fixed values.

---

## **drop()**

**Purpose:**
Removes rows or columns.

**Usage:**
Used in data cleaning.

---

## **groupby()**

**Purpose:**
Groups data based on categories.

**Usage:**
Used for aggregated analysis.

---

## **mean()**

**Purpose:**
Calculates average value.

**Usage:**
Used with groupby() or directly on columns.

---

## **matplotlib.pyplot (plt)**

**Purpose:**
Used for data visualization.

**Usage:**
Helps create graphs and plots.

---

## **plot()**

**Purpose:**
Generates plots such as line, bar, etc.

**Usage:**
Used for visualizing trends in data.

---

## **bar()**

**Purpose:**
Creates bar charts.

**Usage:**
Used for comparing categorical data.

---

## **hist()**

**Purpose:**
Creates histogram plots.

**Usage:**
Used to show data distribution.

---

## **Conclusion**

The notebook uses functions for:

* Data loading and cleaning
* Handling missing values
* Grouping and aggregation
* Data visualization

These functions are essential for analyzing and presenting data effectively.
