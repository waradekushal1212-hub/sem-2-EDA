# README – Functions Used in Experiment 10

## **Overview**

This document describes the key Python and pandas functions used in the notebook for data handling and basic analysis.

---

## **pd.read_csv()**

**Purpose:**
Reads data from a CSV file and loads it into a DataFrame.

**Usage:**
Used to import datasets for analysis.

---

## **pd.DataFrame()**

**Purpose:**
Creates a DataFrame from structured data.

**Usage:**
Used when manually creating datasets or modifying existing data.

---

## **head()**

**Purpose:**
Displays the first few rows of the dataset.

**Usage:**
Helps in quickly inspecting the structure of the data.

---

## **tail()**

**Purpose:**
Displays the last few rows of the dataset.

**Usage:**
Used to verify the end portion of the dataset.

---

## **info()**

**Purpose:**
Provides a summary of the DataFrame.

**Usage:**
Shows column names, data types, and non-null values.

---

## **describe()**

**Purpose:**
Generates statistical summary of numerical columns.

**Usage:**
Used to understand data distribution, mean, standard deviation, etc.

---

## **shape**

**Purpose:**
Returns the dimensions of the DataFrame.

**Usage:**
Used to know the number of rows and columns.

---

## **columns**

**Purpose:**
Lists all column names in the DataFrame.

**Usage:**
Helps in identifying available variables.

---

## **dtypes**

**Purpose:**
Shows the data types of each column.

**Usage:**
Useful for understanding the nature of data.

---

## **isnull()**

**Purpose:**
Detects missing values in the dataset.

**Usage:**
Returns a boolean DataFrame indicating null values.

---

## **sum()**

**Purpose:**
Calculates the sum of values.

**Usage:**
Often used with isnull() to count missing values.

---

## **drop()**

**Purpose:**
Removes specified rows or columns.

**Usage:**
Used for data cleaning.

---

## **fillna()**

**Purpose:**
Replaces missing values with specified values.

**Usage:**
Helps in handling incomplete data.

---

## **sort_values()**

**Purpose:**
Sorts data based on a column.

**Usage:**
Used to arrange data in ascending or descending order.

---

## **Conclusion**

The notebook uses fundamental pandas functions for:

* Data loading
* Data inspection
* Data cleaning
* Basic statistical analysis

These functions are essential for preprocessing and understanding datasets before applying advanced techniques.
